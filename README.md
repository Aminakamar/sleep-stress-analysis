# Sleep, Screen Time and Stress Analysis

## 📌 Objective
To analyze the impact of daily screen time on sleep duration and stress levels.

## 📊 Dataset Features
- Daily Screen Time (hours)
- Sleep Duration (hours)
- Stress Level
- Sleep Quality Score
- Physical Activity
- Mental Fatigue Score

## 📈 Analysis Performed
- Screen Time vs Sleep Duration
- Screen Time vs Stress Level
- Sleep Duration vs Stress Level
- Correlation Analysis

## 📊 Results
- Average Screen Time: 5.5 hours
- Average Sleep Duration: 6.5 hours
- Average Stress Level: 7

## 🎯 Conclusion
Higher screen time is associated with reduced sleep duration and increased stress levels. Managing screen time can help improve overall well-being.

- Python
- Pandas
- Matplotlib
- Google Colab
